# ⚡ Text Repeater Ultra 

A high-performance, futuristic web utility designed to repeat text instantly with custom formatting. Built with a **Holographic Glassmorphism** UI, it offers a premium experience for generating repetitive strings.

![App Preview](https://i.ibb.co/23yBV7LV/image.png)

---

## ✨ Features

- **Holographic UI:** Modern animated mesh background with ultra-blur glass effects.
- **High Performance:** Uses optimized JavaScript `Array.join` for lag-free generation (even for 10,000+ repeats).
- **Smart Toggles:** Interactive switches for adding spaces or new lines seamlessly.
- **One-Tap Actions:**
  - ⚡ **Turbo Run:** Instant text generation.
  - 📋 **Smart Copy:** Copies to clipboard with visual success feedback.
  - 📱 **WhatsApp Share:** Directly send your generated text to any chat.
- **Fully Responsive:** Adapts perfectly to mobile, tablet, and desktop screens.
